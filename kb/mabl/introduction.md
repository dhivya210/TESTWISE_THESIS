# Mabl Introduction

Mabl is an intelligent test automation platform that enables teams to create, execute, and maintain end-to-end tests with minimal coding.

## Key Features

- **Low-code test creation**: Visual test builder
- **Automatic maintenance**: Self-healing tests
- **Cloud-native**: Fully managed test execution
- **Data-driven testing**: Support for test data management
- **API testing**: Combine UI and API tests

## Getting Started

1. **Sign up**: Create account at mabl.com
2. **Create application**: Define your application URL
3. **Install Trainer**: Add Mabl Trainer browser extension
4. **Record tests**: Use Trainer to create test flows
5. **Run tests**: Execute in Mabl cloud or integrate with CI/CD

## Test Creation

Mabl uses a visual test builder:
1. Open Mabl Trainer extension
2. Start recording
3. Perform actions on your application
4. Add assertions and validations
5. Configure test data
6. Save and organize tests
